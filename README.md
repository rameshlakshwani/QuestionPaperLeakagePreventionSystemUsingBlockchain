# 🔐📄 Secure Question Paper Leakage Prevention System using Blockchain ⛓️

## 🌟 Description
This project uses 🔐 Encryption and ⛓️ Blockchain technology to securely store question papers and prevent leaks. Only authorized users 👨‍🏫 can access papers at the scheduled exam time ⏰, ensuring security and transparency.

## 🚀 Features
✅ Secure Question Paper Storage  
✅ Blockchain Verification ⛓️  
✅ Time-Based Access Control ⏰  
✅ Tamper Detection 🚨  
✅ User Authentication 🔑  

## 🛠️ Technologies Used
☕ Java  
🌱 Spring Boot  
🗄️ MySQL  
⛓️ Blockchain  
🔒 AES Encryption  
🔑 SHA-256 Hashing
